# zhihuge-skill

一个面向机器学习、联邦学习、分布式优化和理论研究的论文写作 Skill。

它将论文组织成完整的可验证链条：

> 问题 → 机制 → 方法组件 → 理论结论 → 实验验证 → 适用边界

本 Skill 从三篇相关论文中交叉蒸馏得到，可用于生成论文架构、逐句写作蓝图、主张—证据矩阵，以及检查理论和实验是否闭环。

## 能做什么

- 生成完整的机器学习论文章节结构
- 生成 Abstract 的 9–10 句写作蓝图
- 生成 Introduction 的六段约 18 句结构
- 设计 Related Work 的三条研究线
- 设计 Methodology 的组件与更新逻辑
- 组织收敛、稳定性和泛化理论
- 将理论变量映射为实验和消融
- 生成 Discussion、Conclusion 和 Limitation
- 检查论文中的过度声称和证据缺口
- 从多篇论文中蒸馏作者的写作范式
